## Game Concept

RibaRun is an HTML5 Canvas-based infinite runner that teaches financial literacy through gameplay. Players must manage two critical resources while avoiding obstacles and traps that threaten their financial well-being.

### Objective
Survive as long as possible by balancing your liquidity and savings while navigating through financial obstacles. Make smart decisions to maintain financial health and avoid the traps that lead to economic hardship.

## How to Play

1. Open `index.html` in any modern web browser
2. Use arrow keys to move your character
3. Press Space to start the game
4. Avoid obstacles and collect resources
5. Keep both Liquidity and Savings balanced

## Technologies

- HTML5 Canvas for rendering
- Vanilla JavaScript for game logic
- CSS3 for styling
